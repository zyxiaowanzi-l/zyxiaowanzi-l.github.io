# 🚀 腾讯云人脸识别使用说明

## 📋 概述

本项目已集成腾讯云AI人脸识别服务，提供高精度、稳定的人脸识别功能，替代了本地AI模型。

## ✨ 优势特点

- **高精度识别**：基于腾讯AI实验室先进算法
- **云端服务**：无需下载大型模型文件
- **稳定可靠**：腾讯云基础设施保障
- **功能丰富**：支持检测、比对、搜索等
- **快速响应**：毫秒级识别速度

## 🔑 配置步骤

### 第一步：获取腾讯云API密钥

1. **登录腾讯云控制台**
   - 访问：https://console.cloud.tencent.com/
   - 使用你的腾讯云账号登录

2. **创建API密钥**
   - 进入"访问管理" → "API密钥管理"
   - 点击"新建密钥"
   - 保存 `SecretId` 和 `SecretKey`

3. **开通人脸识别服务**
   - 进入"AI" → "人脸识别"
   - 开通服务并选择计费方式

### 第二步：配置API密钥

1. **编辑配置文件**
   - 打开 `tencent-ai-config.js`
   - 替换以下内容：
   ```javascript
   secretId: 'YOUR_SECRET_ID',        // 替换为你的SecretId
   secretKey: 'YOUR_SECRET_KEY',      // 替换为你的SecretKey
   ```

2. **选择服务地域**
   - 根据你的位置选择最近的地域
   - 推荐：`ap-beijing`（北京）、`ap-shanghai`（上海）

### 第三步：测试服务

1. **刷新网页**
2. **点击"📷 人脸识别验证"按钮**
3. **允许摄像头权限**
4. **拍照测试识别功能**

## 🎯 功能说明

### 人脸检测
- 自动检测图像中的人脸
- 返回人脸位置、属性、质量等信息
- 支持多人脸检测

### 人脸比对
- 比较两张照片中的人脸相似度
- 返回相似度分数（0-100）
- 用于身份验证

### 人脸搜索
- 在人员库中搜索匹配的人脸
- 支持分组管理
- 返回最佳匹配结果

### 人员管理
- 创建人员库
- 添加人员信息
- 管理人脸数据

## 💰 费用说明

### 计费方式
- **按量计费**：按API调用次数收费
- **免费额度**：新用户有免费调用次数
- **价格透明**：详细价格可在控制台查看

### 费用控制
- 设置API调用频率限制
- 监控使用量
- 设置费用告警

## 🔧 故障排除

### 常见问题

1. **API调用失败**
   - 检查API密钥是否正确
   - 确认服务是否已开通
   - 检查网络连接

2. **识别精度低**
   - 确保照片质量良好
   - 光线充足，角度正确
   - 人脸清晰可见

3. **响应速度慢**
   - 检查网络延迟
   - 选择就近的地域
   - 优化图片大小

### 调试方法

1. **查看控制台日志**
   - 按F12打开开发者工具
   - 查看Console标签页的错误信息

2. **检查网络请求**
   - 查看Network标签页
   - 确认API请求是否成功

3. **验证配置**
   - 检查API密钥格式
   - 确认地域设置正确
